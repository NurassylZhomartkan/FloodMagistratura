# Favicon

Эта папка предназначена для хранения файлов favicon сайта FloodSite.

## Как добавить favicon

1. Создайте или скачайте файл `favicon.ico`
2. Поместите файл `favicon.ico` в эту папку: `frontend/public/favicon/`
3. Перезапустите сервер разработки (если он запущен)

## Рекомендуемые размеры (опционально)

Для лучшей поддержки на разных устройствах и платформах рекомендуется иметь следующие файлы:

- **`favicon.ico`** - основной файл (обязательный)
  - Размеры: 16x16, 32x32, 48x48 пикселей
  - Формат: .ico
  
- **`favicon-16x16.png`** - для мобильных устройств (опционально)
  - Размер: 16x16 пикселей
  - Формат: PNG
  
- **`favicon-32x32.png`** - стандартный размер (опционально)
  - Размер: 32x32 пикселей
  - Формат: PNG
  
- **`apple-touch-icon.png`** - для iOS устройств (опционально)
  - Размер: 180x180 пикселей
  - Формат: PNG

**Минимально достаточно поместить только `favicon.ico` в эту папку.**

## Генерация favicon

Если у вас есть изображение и нужно создать favicon, используйте онлайн инструменты:

- https://realfavicongenerator.net/ (рекомендуется)
- https://favicon.io/
- https://www.favicon-generator.org/

Эти инструменты генерируют все необходимые размеры и форматы из одного изображения.

## После добавления файлов

После того как вы поместите файл `favicon.ico` в эту папку:

1. Если сервер разработки запущен, перезапустите его (`npm run dev`)
2. Обновите страницу в браузере (может потребоваться жесткая перезагрузка: Ctrl+Shift+R или Ctrl+F5)
3. Favicon появится во вкладке браузера

## Примечание

Если файл `favicon.ico` отсутствует, браузер просто не будет показывать favicon, но сайт будет работать нормально.
